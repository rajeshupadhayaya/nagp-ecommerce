# Review Collection (NoSQL)

Stores user reviews and ratings for products.

**Example Document:**
```json
{
  "_id": "review_abc",
  "product_id": "product_123",
  "user_id": "user_789",
  "rating": 5,
  "title": "Excellent T-Shirt!",
  "comment": "Great quality and fit. I would definitely recommend it.",
  "created_at": "2026-03-15T14:30:00Z"
}
```
